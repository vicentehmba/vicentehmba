## About me

I’m a software developer interested in exploring new technology and turning it into practical products.

I enjoy working through unfamiliar problems, understanding how emerging tools change what can be built, and finding simpler ways to connect technology with real human needs.

My interests currently sit around AI systems, developer tools, cross-platform applications, product design, and technologies that make software more capable and accessible.

## Current focus

I’m currently building **Zencode**, an AI coding workspace designed to bring different models, agents, tools, terminals, and development workflows into one environment.

At its center is **ZEN**, an orchestration runtime that can work with model-native tools while also supporting external agents through protocols and SDKs such as ACP.

Beyond Zencode, I’m interested in exploring how newer technologies can be applied to overlooked problems, underserved markets, and workflows that existing products still handle poorly.
